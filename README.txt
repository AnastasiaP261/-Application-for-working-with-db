
The client application allows you to send and process requests to a database that represents a set of data about interviews, 
employees, and job seekers from a certain firm. 

Клиентское приложение позволяет отправлять и обрабатывать запросы к базе данных, которая представляет набор данных о 
собеседованиях, сотрудниках и соискателей некоторой фирмы.
